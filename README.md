# 🌟 Todolist App

A beautifully crafted, fully responsive Todo application built with React, Vite, and Chakra UI v3. This app features smooth animations, modern UI elements, and a delightful user experience.

## ✨ Features

- **Add & Edit Tasks**: Easily create new tasks and double-click to edit them.
- **Completion States**: Mark tasks as done—completed tasks automatically glide to the bottom of the list.
- **Smooth Animations**: Powered by `framer-motion` for fluid add, delete, and reorder animations.
- **Stunning UI Elements**: 
  - 🌠 **Meteors** background effect
  - 💧 **Ripple Button** for satisfying interactions
  - ✨ **Glare Hover** effects on task cards
  - 🔠 **Morphing Text** headers
- **Local Storage**: Automatically saves your tasks so you don't lose them when you refresh.
- **Dark/Light Mode**: Toggle seamlessly between themes with the animated theme toggler.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Components**: [Chakra UI v3](https://chakra-ui.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites

Make sure you have Node.js installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate into the project directory:
   ```bash
   cd todolist2
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:5173/` (or the port specified in your terminal).

## 💡 How to Use

- **Add a task**: Type your task into the input field and hit the "Add" button or press Enter.
- **Edit a task**: Double-click on any task text to edit it, then click the checkmark to save.
- **Complete a task**: Check the circle on the left to mark it as done. It will automatically move to the bottom.
- **Delete a task**: Click the red trash icon on the right to remove a task permanently.
